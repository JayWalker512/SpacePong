SpacePong written by Brandon Foltz as practice for coding in C.

Controls:
Up/Down: controls Player 1.
A/Z: controls Player 2 (if enabled).
D: Toggle debug mode.
F: Toggle fullscreen.
B: Load new background image.
P: Pause.
M: Return to menu.
Esc: Quit.
